Advanced filtering block and creating selection sets
=====================================================
.. versionadded:: 0.2.0

``pyautocad`` allows you to create selection sets and filter objects in blocks
with Django ORM-like interface.

It adds function :func:``select`` to :class:``AutocadDocument`` and
:func:``filter`` to all :class:``IAcadObject`` inheritors.
